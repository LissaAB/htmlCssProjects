CSS IDs

An ID is a UNIQUE identifier

ID Notes

  • In CSS, an ID starts with an octothorpe (#)
  • Example use for IDs: Targeting a specific header or section
  • Each ID should only be used once per page - they are unique!

    • This is important with JavaScript later on as well.
  • Use it for one off styling.

IDs versus Classes

IDs

This id will be linked in a CSS page by a # character.

You can't have two ids on an element. If you do, neither will be applied.
